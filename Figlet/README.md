### Installing Figlet in a Mac (need to have brew first)
```bash
brew install figlet
```


### Browse the spiffy font-types
```bash
for line in `figlist`; do figlet -f $line $line; done
```


### Fire Away !: ) 

```bash
figlet -f StarWars "Epic ReadmeBanner"
```
